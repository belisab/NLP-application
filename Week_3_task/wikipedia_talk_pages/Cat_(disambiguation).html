CAT scan

[edit]

In the UK CAT scan stands for Computer Aided Tomography. - It is a type of computer assisted x-ray machine that can generate synthetic 3d images. - also called a brain scanner (generic / informal). Hope that helps.

About disambiguation pages

[edit]
  • The point of a disambiguation page is not to list every possible acronym, definition or use of a particular word; disambiguation pages are meant to assist a Wikipedia user in finding an existing page whose topic is (or could be) the actual word used in their search query.
  • I've removed MANY of the entries on this page because it is doubtful that a person would simply type "cat", "cats", or "CAT" in an attempt to find those specific topics, for example: Combat Application Tourniquet, CAT scan, Common Admission Test, Canadian Army Trophy, Cat Stevens etc...
  • Please see Wikipedia:Disambiguation and Wikipedia:Manual of Style (disambiguation pages) for more info.
